This package contains:

   Magnolia Core 5.4
   Magnolia Internationalization Framework 5.4
   Magnolia Imaging Support Module 3.2
   Magnolia JAAS 5.4
   Magnolia Imaging Module 3.2
   Magnolia Rendering 5.4
   Magnolia Templating 5.4
   Magnolia Templating for JSP 5.4
